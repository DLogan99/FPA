# Modern Industrial Design System

## Palette
- Background: #121417 (charcoal)
- Surface: #151a20 (steel panel)
- Surface-alt: #0f1419 (shadow panel)
- Text: #e5e7eb (off-white)
- Muted: #8b949e (steel gray)
- Accent: #f97316 (safety orange)
- Border: #2b313a (machined edge)

## Typography
- Base: system sans, regular weight for body text.
- Headings: semibold, slightly tighter tracking.
- Data: prefer tabular numbers; use short labels.

## Component standards
- Tabs: raised panel feel with accent active tab and darker inactive tabs.
- Buttons: solid fills with high-contrast text; hover uses accent color.
- Tables: alternating rows, clear header banding, and subtle gridlines.
- Surfaces: keep panels separated with thin borders, avoid heavy shadows.
